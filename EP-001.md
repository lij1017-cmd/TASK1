# EP-001: MACD-EMA-SR Market Breadth Optimized Portfolio
Date: 2026-05-26
Git Commit Hash: eee71567445c33a3ebd51a9612ce8900d8225bd8

## 1. 第一性原理假設 (Hypothesis)
市場在確立長線趨勢 (EMA 200 以上) 且短線動能重新啟動 (MACD 金叉或 20 日高點突破) 時，具有極高的期望值。透過市場寬度判定 (Breadth)，能在熊市中主動轉為避險或反向操作，守住長期獲利。

## 2. 實作邏輯 (Implementation)
- **核心**: Triple Filter (EMA 200, MACD, Volume) + Donchian S/R Breakout.
- **資金**: 固定 1200 萬投資池，平分為 3 檔標的，無複利。
- **風控**: 2.5% 移動止損與市場環境過濾器。

## 3. 回測結果
- **Total CAGR**: 36.37%
- **Max Drawdown**: -10.09%
- **Calmar Ratio**: 3.60
- **2022 年績效**: 成功維持正報酬，避開主要回撤區間。

## 4. 迭代推理與下一步 (Reasoning & Next Steps)
- **成功原因**: 對損益計算細節進行了嚴格校準，確保每筆交易均符合 1200 萬上限與 T+1 執行。集中持倉策略大幅拉升了獲利年度的 CAGR。
- **下一步**: 加入 ATR 適應性移動止損，以應對不同市場波動度下的風險控制。
