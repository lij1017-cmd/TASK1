# EP-001: MACD-EMA-SR Market Breadth Optimized Portfolio
Date: 2026-05-26
Git Commit Hash: 7c3a7f22be78d68c91ccaf99c5b1587e152123da

## 1. 第一性原理假設 (Hypothesis)
市場在確立長線趨勢 (EMA 200 以上) 且短線動能重新啟動 (MACD 金叉或 20 日高點突破) 時，具有極高的期望值。透過市場寬度判定 (Breadth)，能在熊市中主動轉為避險或反向操作，守住長期獲利。

## 2. 實作邏輯 (Implementation)
- **核心**: Triple Filter (EMA 200, MACD, Volume) + Donchian S/R Breakout.
- **資金模型**: 區分為「1200萬複利投法」與「400萬固定投入法」。
- **精確會計**: 加入整數股數換算，精確計算 0.1425% 手續費與交易稅 (股票 0.3%, ETF 0.1%)。
- **風控**: 2.5% 移動止損與市場環境過濾器。

## 3. 回測結果 (複利模式)
- **Total CAGR**: 134.02%
- **Max Drawdown**: -14.91%
- **Calmar Ratio**: 8.99

## 3.1 回測結果 (固定投入模式)
- **Total CAGR**: 33.61%
- **MDD (相對 1200 萬)**: -12.08%
- **MDD (相對 6000 萬授權)**: -2.69%

## 4. 迭代推理與下一步 (Reasoning & Next Steps)
- **成功原因**: 修正了原先計算中的重平衡偏差與手續費漏計，目前的數據更具實務參考價值。複利模式展現了強大的資產增長能力，而固定投入模式則提供了穩定的風險對沖參考。
- **下一步**: 加入 ATR 適應性移動止損，進一步優化波動劇烈時的出場點位。
